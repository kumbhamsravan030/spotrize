# SpotRize Website UI

This repository contains the frontend user interface for the **SpotRize** website. The site is a responsive, modern HTML/CSS/JavaScript web application featuring multiple pages.

## Features
- **Clean & Minimal Design** with an energetic purple gradient aesthetic.
- **Responsive Layout** catering to mobile and desktop experiences.
- **Shared JavaScript Navigation**, dynamically injecting the header, footer, floating chatbot, and social media tabs across 7 different pages without needing a heavy frontend framework.
- **7 Pages Included**: Home (`index.html`), Services, About, Portfolio, Career, FAQ, and Contact.

## How to Run Locally

You can run the web app simply by double-clicking `index.html` in your file browser.

Alternatively, you can run a local web server:

1. Make sure you have Node.js installed.
2. Run the following command:
\`\`\`bash
npm run dev
\`\`\`
3. Open the provided `localhost` link in your browser.
